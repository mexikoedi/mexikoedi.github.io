Description of mexikoedi's Website

This Website represents all objects that mexikoedi created on GitHub and on the Steam Workshop.
It will be updated in the future to add new creations or to improve this website.
Follow the development of this website on GitHub. Look into the Website Code section below.
You can click on one of the white marked creations below to reach the creation on GitHub or the Steam Workshop.
Under the Steam Workshop you can see the Games and further below the Website Code.
Click the blue "mexikoedi" to see his profile on the correspondent platform.
You can see a YouTube Playlist at the bottom of the website with all creations of mexikoedi.
Leave a like and a comment behind. This helps a lot for future projects.
Also don't forget to follow his Steam Workshop profile and his GitHub profile if you like his creations.
You can also click on this Webwiki Link to rate this website. This is very appreciated too.
Thank you for your visit and stay tuned.
